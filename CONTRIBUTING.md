# 🤝 المساهمة في نظام البطولة

شكراً لاهتمامك بالمساهمة في مشروع نظام البطولة!

## 🚀 كيفية المساهمة

### 1. Fork المشروع
- اذهب إلى https://github.com/fronre/system-tournament-ar
- انقر "Fork" لنسخ المشروع

### 2. Clone نسختك
```bash
git clone https://github.com/YOUR-USERNAME/system-tournament-ar.git
cd system-tournament-ar
```

### 3. أنشئ فرع جديد
```bash
git checkout -b feature/your-feature-name
```

### 4. قم بالتغييرات
- أضف ميزات جديدة
- أصلح الأخطاء
- حسّن التصميم

### 5. Commit التغييرات
```bash
git add .
git commit -m "Add your feature description"
```

### 6. Push و Pull Request
```bash
git push origin feature/your-feature-name
```
ثم أنشئ Pull Request على GitHub

## 📝 قواعد المساهمة

1. **كن واضحاً** في رسائل Commit
2. **احترم** تصميم المشروع الحالي
3. **اختبر** تغييراتك قبل الطلب
4. **استخدم** العربية في الواجهة والتعليقات
5. **كن إيجابياً** في التعليقات والمراجعات

## 🏷️ التسمية

- **Features:** `feature/اسم-الميزة`
- **Bug Fixes:** `fix/وصف-الإصلاح`
- **Documentation:** `docs/التحديث`
- **Style:** `style/تحسين-التصميم`
