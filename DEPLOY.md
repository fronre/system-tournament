## 📦 **طريقة النشر (GitHub Pages)**

### 🎯 **الطريقة الأولى: باستخدام السكربت الجاهز**

1. **افتح الطرفية (Terminal)**
2. **نفذ الأمر التالي:**
   ```bash
   npm run deploy
   ```
   أو
   ```bash
   bash deploy.sh
   ```

### 🛠️ **الطريقة الثانية: يدوياً**

1. **تأكد من أنك على الفرع الصحيح:**
   ```bash
   git checkout gh-pages
   ```

2. **أضف كل التغييرات:**
   ```bash
   git add .
   ```

3. **اعمل commit:**
   ```bash
   git commit -m "تحديث الموقع"
   ```

4. **ادفع إلى GitHub:**
   ```bash
   git push origin gh-pages
   ```

---

## 🔧 **إعداد GitHub Pages (مرة واحدة فقط)**

### 📝 **الخطوات:**

1. **اذهب إلى مستودعك على GitHub**
2. **اضغط على Settings**
3. **من القائمة اليسرى، اضغط على Pages**
4. **تحت Source، اختر:**
   - Branch: `gh-pages`
   - Folder: `/ (root)`
5. **اضغط Save**

### 🖥️ **تشغيل الخادم المحلي:**
