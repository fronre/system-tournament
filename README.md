# 🏆 نظام بطولة 2 ضد 2 الاحترافي

نظام إدارة بطولات eSports احترافي بتصميم Neon عصري.

## ✨ المميزات

### 🎮 إدارة اللاعبين
- إضافة وحذف اللاعبين بسهولة
- تخصيص ألوان لكل لاعب
- واجهة سلسة ومتجاوبة

### 🎯 عجلة القرعة
- عجلة تفاعلية بتصميم Canvas
- حركة واقعية مع تأثيرات easing
- اختيار عشوائي دقيق للاعبين

### 🏆 شجرة البطولة
- نظام بطولة كامل (8 فرق)
- الدور الأول → نصف النهائي → النهائي
- تحديد الفائزين بشكل تفاعلي
- احتفال البطل مع شاشة ذهبية

### 🎨 التصميم
- Dark mode احترافي
- تأثيرات Neon و Glow
- رسوم متحركة سلسة
- تصميم متجاوب للهاتف والكمبيوتر

## 🚀 كيفية الاستخدام

1. **إضافة اللاعبين:** أدخل أسماء اللاعبين وانقر "إضافة"
2. **دور العجلة:** عند وجود لاعبين، انقر "دور العجلة" للاختيار
3. **تكوين الفرق:** كل 4 لاعبين يتم إنشاء مباراة 2vs2 تلقائياً
4. **إدارة البطولة:** اختر الفائزين في كل مرحلة للوصول للنهائي
5. **البطل:** حدد بطل البطولة وشاهد احتفال الفوز!

## 🛠️ التقنيات

- **HTML5** - هيكل الصفحة
- **CSS3** - التصميم والأنيميشن
- **JavaScript** - المنطق والتفاعل
- **Canvas API** - رسم العجلة
